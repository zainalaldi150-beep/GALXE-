### 4. **Cara Menjalankan Website**

#### Opsi 1: Menggunakan Script
```bash
# Linux/Mac
./start.sh

# Windows
start.bat
```

#### Opsi 2: Manual
```bash
# Menggunakan Python
python3 -m http.server 8000

# Menggunakan Node.js
npx live-server

# Menggunakan PHP
php -S localhost:8000
```

#### Opsi 3: Langsung
- Buka `index.html` di browser
- Atau buka `run.html` untuk halaman launch
